Credit card fraud is a major threat in today's digital economy, costing billions annually. The aim of this project is to build a machine learning system capable of accurately identifying fraudulent credit card transactions and enhancing customer protection.

üìö Abstract

Credit card fraud involves unauthorized transactions performed by stealing the card physically or accessing it digitally. This project uses machine learning models to detect fraudulent patterns from transaction data, handling challenges like class imbalance and data sensitivity.

- Source: Kaggle Credit Card Fraud Dataset

- Highly Imbalanced: Only 0.17% fraudulent transactions

  • Exploratory Data Analysis
  • Model Building (Logistic Regression, Decision Tree, Random Forest)
  • Model Evaluation (focused on Recall)
  • Flask App Deployment

- Special focus was given to minimizing false negatives (missing fraud cases).
